# 番茄钟 Pomodoro Timer

一个简洁的桌面番茄钟，单文件 HTML，双击即用。

## 功能

- 专注模式（25分钟）/ 短休（5分钟）/ 长休（15分钟）
- 可视化环形进度倒计时
- 自动循环：专注 → 短休 → 专注 → ... → 长休
- 声音提醒（Web Audio 合成，无需音频文件）
- 浏览器桌面通知
- 完成番茄数 & 总专注时长统计
- 可自定义时长和长休间隔
- 键盘快捷键：`空格` 开始/暂停、`R` 重置、`S` 跳过
- 数据本地持久化（localStorage）

## 使用

双击 `pomodoro.html` 在浏览器中打开即可。首次点击页面会请求通知权限。
